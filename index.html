Uptime Monitoring API

A RESTful API that monitors user-defined URLs and sends SMS if they are up or down. To use this API, users need to register themselves first. After registration, they need to get a token and use it wherever required. Then users need to register the URLs that they need to monitor. The API checks the URLs every minute and sends SMS if the URLs' status changes to up or down. This API is written in Raw Node.
